# 📖 About The Project

This project is a modern **Training Academy Landing Page** designed using **HTML5**, **CSS3**, and **Bootstrap 5**.

The website features a clean UI, responsive layout, Bootstrap components, course cards, testimonials, statistics, admission form, and footer section.

It is fully responsive and optimized for desktop, tablet, and mobile devices.
